1. разница между библиотекой и фреймворком

по сути библиотека - набор готовых решений, функций или классов, которые дергаются кодом программиста

Фреймворк — это готовый каркас для приложения, который управляет кодом разработчика.
В отличие от библиотеки, фреймворк определяет структуру приложения и управляет его жизненным циклом.

3. чем отличаются стратегии генерации ПК

GenerationType.AUTO - (автоматическое определение взависимости от диалекта) id генерится в hibernate
GenerationType.IDENTITY - (сиквенс генерится базой данных при указании первичного ключа) id генерится базой данных
GenerationType.SEQUENCE - (сиквенс генерится хибером при создании таблиц) - используется последовательность – специальный объект БД для генерации уникальных значений.
GenerationType.TABLE - (создается отдельная таблица, в которой хранится последнее значение id) для генерации уникального
значения используется отдельная таблица, которая эмулирует последовательность. Когда требуется новое значение,
JPA провайдер блокирует строку таблицы, обновляет хранящееся там значение и возвращает его обратно в приложение.
Эта стратегия – наихудшая по производительности и ее желательно избегать.
