В ЦЕРН в 1991 году разработали экспериментальную версию протокола HTTP 0.9.
Первая открытая версия HTTP 1.0 появилась в 1996 году, она описана в документе RFC 1945.

Почти сразу после HTTP 1.0, в 1997 году, вышла обновленная версия протокола HTTP 1.1.
В этой версии добавили кэширование, постоянное соединение, аутентификацию и некоторые
другие возможности. Документы RFC, описывающие HTTP 1.1, несколько раз обновлялись,
сейчас действует RFC 9112 от 2022 года. Версия HTTP 1.1 используется сейчас.

В 2015 году вышла версия HTTP/2 (описана в RFC 9113), основная задача которой – повышение
производительности. Самая последняя версия HTTP/3, описанная в RFC 9114, использует на
транспортном уровне протокол QUIC вместо TCP.

HTTP/1.1 – поддерживается всеми браузерами, но используется реже, если сервер поддерживает более новые версии.
HTTP/2 – стандарт с 2015 года, используется по умолчанию на большинстве сайтов.
HTTP/3 – новый протокол на основе QUIC, поддерживается всеми современными браузерами, но требует, чтобы сервер его тоже поддерживал.
